== Description ==
 
UniForms extends WordPress with light weight drag & drop contact form builder. Easily build your contact page with simple & light drag& drop editor.

= Docs & Support =

We are writing documentation and it should be available quickly, remeber that this is our first release of UniForms and you should contribute if you find any bugs or issues.

A basic documentation can be found on next website (bottom of page):
[UniForms Basic Setup](https://theme-vision.com/plugins/uniforms/)
 
== Installation ==
 
1. Upload `uniforms` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
